Format A (left), Format B (right)

1. 101 1110

Format A:
```
Bits = 101 1110

e = 5
E = e - bias = 2
f = .1110 = 7/8
M = 1+f = 15/8

Value = 15/8 * 2^2 = 15/2
```

Format B:
```
Num = 111.10
e = E + bias = 9 = 1001

f = 111

Bits = 1001 111
Value = 15/2
```

2. 010 1001

Format A:
```
Bits = 010 1001

e = 2
E = e - bias = -1
f = .1001 = 9/16
M = 1+f = 25/16

Value = 25/16 * 2^-1 = 25/32
```

Format B:
```
e = E + bias = 6 = 0110
f = round(.1001, 3) -> .100 = 1/2

Bits = 0110 100
Value = 3/2 * 2^-1 = 3/4 = 24/32
```

3. 110 1111

Format A:
```
Bits = 110 1111

e = 6
E = e - bias = 3
f = 1111 = 15/16
M = 1+f = 31/16

Value = 31/16 * 2^3 = 31/2
```

Format B:
```

f = 000
e = E + bias + 1 = 11 = 1011

Bits = 1011 000
Value = 2^4 = 16
```
